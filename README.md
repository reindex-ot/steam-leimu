# Steam GBC

基于 [Millennium](https://github.com/SteamClientHomebrew/Millennium) 框架的 Steam 皮肤

![preview](https://github.com/user-attachments/assets/3bc2da0b-9ab3-450c-b1cf-0b9234803505)

## 功能

- MD3 风格圆角设计
- 深色/明亮主题切换
- 三种圆角大小（小/中/大）
- 自定义背景图片
- 视频背景功能

## 安装

1. 安装 [Millennium](https://github.com/SteamClientHomebrew/Millennium)
2. 将皮肤文件夹放入 `steamui/skins/` 目录
3. 在 Millennium 设置中选择皮肤

## 自定义背景

通过 `skintool.exe` 可实现自定义背景，调整透明度（0~1）和修改皮肤名称，背景图片推荐分辨率 1920×1080
